# 介绍
此项目包含发行中常见的API，包括对内和对外接口。

- 手游（安卓&iOS）SDK自营通信接口
- 手游（安卓&iOS）SDK联运通信接口
- HTML5游戏SDK自营通信接口
- HTML5游戏SDK联运通信接口
- 官网&活动接口
- 等
